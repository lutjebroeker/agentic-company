# Offer Stack — Hormozi Value Equation

Value = (Dream Outcome × Perceived Likelihood) ÷ (Time Delay × Effort)

## Gold — Done-for-You VPS Setup
- **Prijs:** €1.497 (launch) / €1.997 (regulier)
- **Deliverable:** Complete installatie op Hetzner VPS, geconfigureerd op klant-doelen, 30 dagen Telegram support
- **Tijdsinvestering:** 5–7 uur per klant
- **Dream outcome:** Systeem live binnen 5 werkdagen, klant hoeft niks te installeren
- **Inclusief:** n8n + Obsidian vault + AI-agent configuratie + onboarding call

## Silver — Managed Check-in
- **Prijs:** €397/maand (alleen voor Gold-klanten)
- **Deliverable:** Maandelijkse systeem-review call (30 min), workflow updates, priority support
- **Tijdsinvestering:** 2 uur per klant per maand
- **Dream outcome:** Systeem blijft werken en groeit mee

## Bronze — Template Pack
- **Prijs:** €247 (eenmalig)
- **Deliverable:** n8n workflow exports (JSON), Obsidian vault structuur, video walkthrough
- **Tijdsinvestering:** Eenmalig 8 uur bouwen, daarna 0
- **Dream outcome:** Zelf aan de slag met bewezen templates

## Revenue model
- Target: €4.500/maand in maand 6
- Pad: 3 Gold-klanten = €4.491 (launch pricing)
- Of: 2 Gold + 2 Silver + 2 Bronze = €4.482
- Schaalbaarheid: Bronze = passief inkomen, Gold = high-touch
