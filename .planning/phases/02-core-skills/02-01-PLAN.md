  <name>Task 2: Bouw n8n workflow "02-01-telegram-listener"</name>
  <files>n8n workflow (via n8n MCP — geen lokaal bestand)</files>
  <action>
    Gebruik de n8n MCP om de workflow aan te maken met de volgende node-structuur:

    **Node 1 — Telegram Trigger**
    - Type: n8n-nodes-base.telegramTrigger
    - Updates: message (alleen tekst, geen media nodig voor MVP)
    - Credentials: Telegram bot (productie-bot token)

    **Node 2 — Code: Extract + NLP routing**
    - Type: n8n-nodes-base.code
    - Taal: JavaScript
    - Logica:
      ```javascript
      const message = $json.body.message.text.toLowerCase();
      const chatId = $json.body.message.chat.id;
      const messageId = $json.body.message.message_id;
      const originalMessage = $json.body.message.text;

      const skillKeywords = {
        ceo:        ['strategie', 'prioriteit', 'bottleneck', 'focus', 'besliss', 'hormozi', 'omzet'],
        marketing:  ['content', 'post', 'linkedin', 'schrijf', 'tekst', 'campagne', 'trust formula'],
        sales:      ['lead', 'dm', 'outreach', 'klant', 'pipeline', 'drift', 'meet', 'prospect'],
        ops:        ['n8n', 'workflow', 'vps', 'server', 'automatiseer', 'deploy', 'technisch'],
        personal:   ['week', 'check', 'accountab', 'doel', '12wy', 'executie', 'percentage'],
        review:     ['review', 'feedback', 'beoordeel', 'score'],
        learn:      ['leer', 'wins', 'fails', 'synthesize', 'verbetering', 'improvements'],
        ship:       ['publiceer', 'deploy', 'ship', 'push', 'verstuur', 'live']
      };

      let detectedSkill = null;
      let maxMatches = 0;
      let allMatches = {};

      for (const [skill, keywords] of Object.entries(skillKeywords)) {
        const matches = keywords.filter(kw => message.includes(kw)).length;
        allMatches[skill] = matches;
        if (matches > maxMatches) {
          maxMatches = matches;
          detectedSkill = skill;
        }
      }

      // Vind tweede hoogste match voor tie-detection
      const sortedMatches = Object.entries(allMatches).sort(([,a],[,b]) => b-a);
      const topTwo = sortedMatches.slice(0, 2);
      const tie = topTwo[0][1] === topTwo[1][1] && topTwo[0][1] > 0;

      const confident = maxMatches >= 1 && !tie;

      // Bouw top-kandidaten lijst voor ambiguity message
      const topCandidates = tie ? topTwo.map(([s]) => `/${s}`).join(' of ') : null;

      return [{ json: { chatId, messageId, originalMessage, detectedSkill, confident, topCandidates } }];
      ```

    **Node 3 — IF: Confident match?**
    - Type: n8n-nodes-base.if
    - Condition: `{{$json.confident}}` is true
    - TRUE path → Node 4 (SSH)
    - FALSE path → Node 5 (Ambiguity reply)

    **Node 4 — Code: Bouw SSH command**
    - Type: n8n-nodes-base.code
    - Logica:
      ```javascript
      const skill = $json.detectedSkill;
      const userMessage = $json.originalMessage.replace(/`/g, "'").replace(/"/g, '\\"');
      const skillPath = `/root/.claude/commands/${skill}.md`;

      // BELANGRIJK: gebruik het volledige pad dat in Task 1 is vastgesteld
      // Vervang /usr/local/bin/claude met het werkelijke pad van 'which claude'
      const command = `SKILL_PROMPT=$(cat ${skillPath})
/usr/local/bin/claude --no-permissions-prompt -p "$SKILL_PROMPT

Vraag: ${userMessage}"`;

      return [{ json: { ...$json, sshCommand: command } }];
      ```

    **Node 5 — SSH: Roep Claude aan**
    - Type: n8n-nodes-base.ssh
    - Credentials: SSH credentials (aangemaakt in Task 1)
    - Command: `{{$json.sshCommand}}`
    - Verwerk output: stdout = Claude's antwoord

    **Node 6 — Code: Formatteer antwoord**
    - Type: n8n-nodes-base.code
    - Logica:
      ```javascript
      const response = $json.stdout || 'Er is iets misgegaan. Probeer opnieuw.';
      const truncated = response.substring(0, 4096); // Telegram max
      return [{ json: { ...$json, formattedResponse: truncated } }];
      ```

    **Node 7 — Telegram: Stuur antwoord**
    - Type: n8n-nodes-base.telegram
    - Operation: sendMessage
    - Chat ID: `{{$node["Code: Extract + NLP"].json.chatId}}`
    - Text: `{{$json.formattedResponse}}`
    - Reply to Message ID: `{{$node["Code: Extract + NLP"].json.messageId}}` (reply in thread)

    **Node 8 — Telegram: Ambiguity reply** (FALSE-pad van Node 3)
    - Type: n8n-nodes-base.telegram
    - Operation: sendMessage
    - Chat ID: `{{$json.chatId}}`
    - Text: `Bedoel je {{$json.topCandidates || '/ceo of /sales'}}? Formuleer je vraag iets specifieker.`
    - Reply to Message ID: `{{$json.messageId}}`

    Activeer de workflow na aanmaken. Gebruik n8n MCP tool `activateWorkflow`.
