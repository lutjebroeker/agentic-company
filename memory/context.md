## IP-modellen

### DRIFT (probleem-model)
Legt uit waarom doelen wegzinken:
- **D**oel stellen — een wens met een deadline
- **R**outine wegzakken — week 3: actie verdwenen
- **I**ntentie ≠ Actie — het gevaarlijkste gat
- **F**eedback ontbreekt — gevoel ≠ data
- **T**olerantie voor uitstel — te aardig = gevaarlijk

### MEET (oplossings-model)
Hoe de Execution Engine werkt:
- **M**eten — elke avond in percentages
- **E**erlijk confronteren — geen excuses, alleen feiten
- **E**valueren — wat zeggen de cijfers?
- **T**erugkoppelen — elke cyclus beter

**Elevator pitch:** "De meeste mensen driften weg van hun doelen — het DRIFT-patroon. De Execution Engine doorbreekt dat via de MEET-cyclus."
